# Задание

- Обработка строк

Разделить строку на части длиной не более substringLength с помощью символа перевода строки. 
Строку делить на части только по разделителю слов. 
Если длина слова превышает substringLength, то оставить слово без изменений.

 - Обработка слов

Удалить слова, первая буква которых встречается в них еще хотя бы один раз.

- Проверка соответствия строки регулярному выражению

Химическая формула вещества, в состав которого входят только водород, углерод и кислород. Примеры: H2O, C2H5OH, CO2.
