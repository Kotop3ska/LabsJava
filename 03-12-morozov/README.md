# Задание

Просуммировать элементы с четными значениями и перемножить элементы с нечетными значениями. 
Если сумма больше, то обнулить элементы с положительными значениями. 
Если произведение больше, то обнулить элементы с отрицательными значениями. 
В случае равенства суммы и произведения вывести соответствующее сообщение.
